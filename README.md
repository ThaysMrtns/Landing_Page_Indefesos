## Getting Started

Para iniciar o servidor de desenvolvimento, execute o comando:

```bash
npm run dev
# or
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o projeto no ambiente de desenvolvimento.
